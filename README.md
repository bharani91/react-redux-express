# React Redux Express
**Another React starter kit that uses Redux, React Router, HMR, SCSS & Express**

------------------

## Usage

**Install**
```
npm install
```

**Start development server**
```
npm start
```

**Build for production**
```
npm run build
```

**Start production server**
```
npm run build:start
```
